To run the program:
---------------------------------------------------------------------------------------------------
1. Import the following files to the folder where these two source codes are run from
	a. train.txt
	b. test5.txt
	c. test10.txt
	d. test20.txt
2. Run
	a. For Conventional Pearson
		Command: python pearson_run.py 120
		Where 120 is the value of K.

	b. For Pearson with IUF
		Command: python pearson_iuf.py 120
		Where 120 is the value of K.

	a. For Pearson with Case Amplification
		Command: python pearson_case.py 120
		Where 120 is the value of K.
--------------------------------------------------------------------------------------------------
