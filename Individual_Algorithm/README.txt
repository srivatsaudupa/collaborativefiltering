To run the program:
---------------------------------------------------------------------------------------------------
1. Import the following files to the folder where these two source codes are run from
	a. train.txt
	b. test5.txt
	c. test10.txt
	d. test20.txt
2. Run
Basic cosine run Command: python cosine_run.py 100
Where 100 is the value of K.
	Produces three files
	a. cosine_result5.txt
	b. cosine_result10.txt
	c. cosine_result20.txt
Item Based Run Command: python cosine_item_run.py 100
	Produces three files
	a. item_result5.txt
	b. item_result10.txt
	c. item_result20.txt
Run combined: python combine_cosine_user_item.py
	Produces three files
	a. result5.txt
	b. result10.txt
	c. result20.txt
---------------------------------------------------------------------------------------------------
